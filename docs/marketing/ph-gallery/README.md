# Product Hunt Gallery Assets

All images in this folder are prepared at `1270 x 760`, the Product Hunt gallery ratio.

Recommended upload order:

1. `01-hero-demo.gif` - first gallery asset; lightweight animated hero.
2. `02-workflow.png` - explains the Harness Agent workflow.
3. `04-feishu-proof.png` - shows the Feishu approval comparison: template matching, 3/3, 18 seconds, zero AI calls.
4. `03-mcp-34-tools.png` - shows the MCP Server with 34 automation tools.

Optional:

- `01-hero-still.png` - static fallback if a platform does not accept GIF.

Notes:

- Do not upload screenshots that include real customer data, private URLs, account names, tokens, or approval contents.
- Replace `docs/media/demo.gif` after recording the real Feishu approval demo if you want the GitHub README to show the same motion asset.
